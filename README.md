Customer Support Router Dashboard

A visually stunning, high-resilience AI Triage and Dispatch Console built for the Software Engineering Internship Assessment. This application intercepts customer complaints, performs real-time sentiment/severity analysis, and drafts empathetic responses utilizing the Anthropic Claude API (Haiku) with full local fallback redundancy.

🚀 Quick Start & Setup

This is a zero-dependency, single-file application. You can run it instantly without compiling any assets or installing packages.

Clone this repository:

git clone https://github.com/CODEalvii/customer-support-router-dashboard

Run the Dashboard:

Simply open index.html in any modern web browser.

VS Code Users: Right-click index.html and select "Open with Live Server".

Run Triage:

Paste your Anthropic API Key (sk-ant-...) in the secured credentials field to run live Claude routing, or leave it blank to instantly experience our high-uptime mock sandbox environment.
